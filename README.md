### Decomposition:
1) Create a list of songs with their details (title, duration, chart ranking etc.)
2) Ask user to choose a key for sorting
3) check if the key chosen is an option for sorting
4) use merge sort to sort the songs according to the key
5) output the sorted list of songs to the user

### Abstraction:
- details like the duration, title, ranking
- 3 </= # of songs </= 15

### Pattern Recognition:
